# IFileDialog
Example for VisualFBEditor

This FreeBASIC code implements an advanced file open dialog with the following main features:

1. Uses the Windows COM interface IFileOpenDialog to create a modern file selection dialog
2. Supports multiple file selection (FOS_ALLOWMULTISELECT)
3. Adds custom control groups:

```
1). Line ending format selection: Windows (CRLF), Linux (LF), Mac OS (CR)
2). Text encoding selection: ASCII, UTF-8, Unicode (UTF-16 LE), Unicode Big Endian
```

5. Implements complete IFileDialogEvents and IFileDialogControlEvents event handling mechanisms
6. Provides detailed event logging output for debugging and monitoring dialog behavior
7. Supports standard functions such as setting initial paths and file type filters
8. Includes comprehensive resource management and memory release mechanisms
